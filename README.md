# 🩺 AI-Based Early Disease Prediction using Machine Learning & Explainable AI (XAI)

An intelligent healthcare prediction system that predicts multiple diseases such as Heart Disease, Diabetes, and Breast Cancer using Machine Learning models with Explainable AI support.

This project helps users get early risk prediction through a simple web interface while also providing transparency behind predictions using XAI techniques.

---

## 🚀 Features

- Multi-Disease Prediction System
- Heart Disease Prediction
- Diabetes Prediction
- Breast Cancer Prediction
- User-Friendly Web Interface
- Machine Learning Based Risk Analysis
- Explainable AI (XAI) Support
- SQLite Database Integration
- Responsive Frontend using HTML/CSS
- Flask Backend

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/AI-Disease-Prediction-System.git
```

### 2️⃣ Navigate to Project Folder

```bash
cd AI-Disease-Prediction-System
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the Flask App

```bash
python app.py
```

### 5️⃣ Open in Browser

```bash
http://127.0.0.1:5000
```
